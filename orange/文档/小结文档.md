这是一个操作系统相关的项目,目前主要包含以下几个部分:

1. 引导启动部分:

- boot.S 和 boot.c 构成系统引导部分
- boot.h 作为头文件(当前为空)
- 在 boot.c 中定义了一个简单的字符数组 "wenzhuzy"

2. 构建系统:

- 使用 Makefile 管理编译流程
- 使用交叉编译工具链(x86_64-elf-)
- 编译选项包括:32位模式、禁用PIE、禁用栈保护等
- 生成二进制镜像文件 disk.img

3. 启动配置:

- start.bash 提供了 QEMU 启动脚本
- 配置了 128MB 内存
- 使用 raw 格式的磁盘镜像文件

### 主要特点

1. 这是一个基础的操作系统开发环境搭建
2. 使用 QEMU 作为虚拟机测试环境
3. 支持交叉编译到 x86 平台
4. 包含基本的引导程序结构
